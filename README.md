## Cách lấy toàn bộ Source Sode và cách chạy chương trình
**1. Cách lấy toàn bộ Source Code:**
```bash
git clone https://github.com/TheAnh-05-UIT/NT101-AnToanMangMayTinh.git
```
**2. Cách chạy chương trình:**
Sử dụng Terminal để chạy các câu lệnh sau: (thay đổi nội dung trong dấu <>)
```bash
cd <Ten thu muc (ví dụ: Lab01)>
```

- Đối với file code C++:
```bash
g++ <tenfile.cpp> -o <tenfile>
./tenfile
```
- Đối với file code Python:
```bash
python <tenfile>.py
```
